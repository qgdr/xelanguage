## 函数

```

```

### 函数返回

```
fn plus_five(x:i32) -> i32 {
    return x + 5;
}

fn main() {
    let x = plus_five(5);

    println!("The value of x is: {}", x);
}
```

## 特殊类型

`None`
`Any`

这两个并不相同，`None` 是一种类型，而 `Any` 可以匹配任何类型，类似于 C 中的 `void`
```rust
fn foo(x: MyType) -> Any
{
    ...
}

fn main(){
    let m1 : MyType <| MyType::new();
    let m2 : MyType <| MyType::new();


    let m2 <| foo(m1);
}
```
这是允许的，不改变 `m2` 的值
